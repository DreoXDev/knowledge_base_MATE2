# Sources

Questa cartella raccoglie fonti e materiali di partenza.

## Sottocartelle

- `official_slides/`: slide o dispense ufficiali.
- `official_exercises/`: esercizi e temi ufficiali.
- `professor_notes/`: note fornite dal professore.
- `student_notes/`: appunti personali o di altri studenti.

## Regola

Non modificare le fonti originali. Se serve una versione trascritta o pulita, inserirla in `03_transcriptions/`.
