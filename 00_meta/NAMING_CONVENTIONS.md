# Naming Conventions

## File e cartelle

- Usare nomi in minuscolo.
- Separare parole con underscore.
- Evitare spazi nei nuovi file Markdown.
- Usare nomi descrittivi e stabili.

## Note teoriche

Formato consigliato:

```text
argomento_specifico.md
```

Esempi:
- `campi_conservativi.md`
- `integrali_di_linea_seconda_specie.md`
- `teorema_di_green.md`

## Link Obsidian

Usare link wiki:

```md
[[green]]
[[campi_conservativi]]
```

Se il nome non e' ancora definitivo, lasciare un TODO vicino al link.
