# Checklist discorsi orali

## Green

- [ ] So enunciare Green.
- [ ] So spiegare l'orientazione positiva.
- [ ] So fare la dimostrazione con le due identita'.
- [ ] So collegarlo a Stokes.
- [ ] So fare un esempio.

## Lagrange

- [ ] So spiegare intuizione geometrica.
- [ ] So scrivere il caso a un vincolo.
- [ ] So scrivere il caso a due vincoli.
- [ ] So dire che da' candidati, non sufficienza.
- [ ] So fare esempio su sfera.

## Stokes

- [ ] So enunciare Stokes.
- [ ] So definire rotore.
- [ ] So spiegare orientazione compatibile.
- [ ] So collegarlo a Green.
- [ ] So fare esempio emisfero/piano.

## Gauss

- [ ] So enunciare Gauss.
- [ ] So definire divergenza.
- [ ] So spiegare normale uscente.
- [ ] So fare idea di dimostrazione per componenti.
- [ ] So fare esempio sfera.

## Campi conservativi

- [ ] So definire potenziale.
- [ ] So dimostrare indipendenza dal cammino.
- [ ] So spiegare circuitazione nulla.
- [ ] So spiegare Poincare.
- [ ] So fare controesempio campo angolare.

## Integrali linea I specie

- [ ] So definire tramite somme.
- [ ] So spiegare $d\ell$.
- [ ] So derivare formula parametrica.
- [ ] So dire che non dipende dall'orientazione.
- [ ] So fare esempio.

## Integrali linea II specie

- [ ] So definire con componente tangenziale.
- [ ] So scrivere forma differenziale.
- [ ] So spiegare orientazione.
- [ ] So collegarlo al lavoro.
- [ ] So collegarlo ai campi conservativi.
